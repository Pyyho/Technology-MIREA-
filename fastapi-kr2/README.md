# Контрольная работа №2 по FastAPI

## Описание проекта

Проект содержит решение всех заданий контрольной работы №2 по курсу "Технологии разработки серверных приложений".

## Выполненные задания

### Задание 3.1 - Создание пользователя
- POST `/create_user` - создание пользователя с валидацией данных

### Задание 3.2 - Работа с продуктами
- GET `/product/{product_id}` - получение продукта по ID
- GET `/products/search` - поиск продуктов по параметрам

### Задание 5.1 - Базовая аутентификация
- POST `/login` - вход в систему
- GET `/user` - защищенный маршрут

### Задание 5.2 - Подписанные сессии
- POST `/login/v2` - вход с подписанной сессией
- GET `/profile` - защищенный профиль с проверкой подписи

### Задание 5.3 - Сессии с временем активности
- POST `/login/v3` - вход с сессией активности
- GET `/profile/v3` - профиль с проверкой времени активности

### Задание 5.4 - Работа с заголовками
- GET `/headers` - получение заголовков запроса

### Задание 5.5 - Модель заголовков
- GET `/headers/v2` - заголовки через Pydantic модель
- GET `/info` - информация с дополнительными заголовками
